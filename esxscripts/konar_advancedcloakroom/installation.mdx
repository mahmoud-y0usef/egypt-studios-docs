# installation
### [esx][paid] konar_advancedcloakroom
* Welcome, I'm so glad you bought my script and I'm glad you joined our https://discord.gg/2Xhag8k8Pm

* in esx_policejob / client.lua

* add

```c
RegisterNetEvent('esx_policejob:playerInService')
AddeventHandler('esx_policejob:playerInService', function()
	playerInService = true
end)
RegisterNetEvent('esx_policejob:playeroutService')
AddeventHandler('esx_policejob:playeroutService', function()
	playerInService = false
end)
```  


* in konar_policearmour 
    * run items.sql
    * in server.cfg
    * start konar_advancedcloakroom
    * start konar_policearmour
